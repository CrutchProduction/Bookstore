## Использование

>### **Bookstore**
В меню "Новая книга" можно создавать новую книгу либо с рандомными данными, либо со своими данными.
Если книжные полки заполнены/нет подходящей книжной полки, то _книга уничтожается_.
<br/><br/>
В меню "Магазин" отображаются все текущие книжные полки, их жанры, возможность поиска книги по названию и Id, а также текущий баланс магазина. <br/>
При нажатии на любую из книжных полок, будет показано её содержимое, а также возможность продать выбранную книгу. При продаже всего содержимого книжной полки, жанр книжной полки будет сброшен и может быть заново присвоен при добавлении новой книги<br/><br/>
Для поиска требуется выйти из книжной полки и ввести название/Id книги в соответствующее поле в нижней части окна, после чего нажать "Enter". В случае успешного поиска, откроется книжная полка, содержащая в себе требуемую книгу, также требуемая книга автоматически выделится, либо, в случае провала, ничего не произойдёт.

>### **ClassLibraryBookstore**
ClassLibraryBookstore - библеотека классов к визуальной части проекта Bookstore
>#### **Класс Shop**
Класс Shop содержит в себе следующие внешне-доступные функции:
```
void AddBook(string name, string author, string genre, int pageAmount, int price) — Добавление книги в магазин с заданными параметрами

void AddBook() — Добавление книги в магазин с рандомными параметрами

void CheckBookName(Book book) — Проверка на повторное название у книги

BookShelf FindBookShelf(string genre) — Поиск книжной полки по жанру, если таковой нет, возвращает либо пустую полку, либо null

void AddBalance(int income) — Добавить деньги к текущему балансу

BookShelf[] GetShelfs() — Возвращает все книжные полки в магазине

int GetBalance() — Возвращает текущий баланс

int GetLastBookId() — Возвращает Id следующей книги
```

>#### **Класс BookShelf**
Класс BookShelf содержит в себе следующие внешне-доступные функции:
```
void AddBook(Book book) — Сохраняет книгу в полке

void RemoveBook(Book book) — Удаляет книгу из полки

Book SearchBook(string name) — Поиск книги по названию, возвращает либо книгу, либо null

Book SearchBook(int id) — Поиск книги по Id, возвращает либо книгу, либо null

int SellBook(Book book) — Продаёт книгу, возвращает стоимость книги

Book[] GetBooks() — Возвращает все книги в полке

string GetGenre() — Возвращает текущий жанр полки
```

>#### **Класс Book**
Класс Book содержит в себе следующие внешне-доступные функции:
```
void SetCopyId(int copyId) — Устанавливает индекс копии книги

string GetName(bool withCopyId) — Возвращает имя книги с или без индекса копии книги

string GetAuthor() — Возвращает автора книги

string GetGenre() — Возвращает жанр книги

int GetId() — Возвращает Id книги

int GetPageAmount() — Возвращает количество страниц в книге

int GetPrice() — Возвращает цену книги
```
